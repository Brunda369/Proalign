# 🚀 ProAlign AI  
### AI-Powered Interview, Posture & Emotion Training Platform

ProAlign AI is an intelligent web-based platform designed to help students and job aspirants improve their interview performance using AI-driven analysis.

It combines:
- 🎤 AI Mock Interviews
- 🧍 Smart Posture Analysis
- 😊 Emotion & Facial Expression Training
- 📊 Professional Performance Dashboard

---

## 🌟 Features

### 🎤 AI Mock Interview
- Real-time question-answer simulation
- Technical answer scoring
- Integrated emotion & posture tracking
- Detailed performance report

### 🧍 Posture Practice
- Live camera posture detection
- Smart posture scoring (/10)
- Voice-based correction feedback
- Session history tracking

### 😊 Emotion Training
- Facial emotion detection
- Confidence score calculation
- Emotional consistency analysis
- Session history tracking

### 📊 Performance Dashboard
- Interview report history
- Posture session history
- Emotion session history
- Professional Confidence Index
- Level system (Beginner → Interview Ready)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

git clone [https://github.com/Brunda369/Proalign.git] 
cd ProAlign-AI

---

### 2️⃣ Create Virtual Environment

python -m venv venv

Activate:

Windows:
venv\Scripts\activate

Mac/Linux:
source venv/bin/activate

---

### 3️⃣ Install Dependencies

pip install -r requirements.txt

---

### 4️⃣ Run the Application

python app.py

Open in browser:

http://127.0.0.1:5000

## 📈 Professional Confidence Index Formula

Professional Index =
(Interview Average × 0.5)
+ (Posture Average × 2.5)
+ (Emotion Average × 2.5)

This creates a balanced professional readiness score out of 100.

---

## 🏆 Level System

Score Range → Level

85+ → 🏆 Interview Ready  
70–84 → 🚀 Advanced  
50–69 → 📈 Improving  
Below 50 → 🌱 Beginner
